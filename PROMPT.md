# Claude Code 開始用プロンプト

以下をコピーして、このフォルダで起動したClaude Codeに最初に貼ってください。

---

食材を3〜5つ選ぶと、一致度の高いレシピをおすすめしてくれる個人用PWAアプリ「レシピヘルパー」を開発します。

まず以下を順に行ってください:

1. `CLAUDE.md`、`docs/REQUIREMENTS.md`、`docs/DECISIONS.md` を読んで、プロジェクトの前提と決定事項を把握してください
2. 5カテゴリ(野菜/肉・魚/乳製品[卵含む]/加工食品/主食・粉類)それぞれの食材マスタ(選択肢として出す食材の正式名称リスト)の案を提示してください
3. 食材マスタが確定したら、レシピデータのJSONスキーマを提示してください
4. その後、開発全体のステップ一覧を提示してください

重要な進め方のルール:
- ステップ一覧を最初に全部見せてから、1ステップずつ私の確認を取って進めてください
- 確認なしに次のステップへ進まないでください
- レシピデータは件数が多いため、ジャンルごとに少量ずつ作成し、都度サンプルを見せて品質を確認させてください(一度に全件作らないでください)
- **件数よりマッチング精度(食材マスタとのタグ整合性)を優先してください**
- Gitのコミットやプッシュも、実行前に必ず確認を取ってください

まずは上記1〜2(ドキュメントの確認と食材マスタ案の提示)から始めてください。

---

## 補足: 途中から再開する場合のプロンプト例

「前回の続きです。CLAUDE.mdとdocs/を読み直して、現在の進捗状況(食材マスタ・レシピ件数・実装済み画面)を確認してから、次のステップを提示してください。」
